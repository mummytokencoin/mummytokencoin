# MUMMYTOKENCOIN - LHOPE Memorial NFTs

## Project Overview
LHOPE Memorial NFTs is a decentralized platform on BSC Mainnet that allows users to create permanent blockchain tributes honoring loved ones affected by liver disease.

## Live DApp
🔗 **Mint your memorial NFT:** https://mint.mummytokencoin.com

## Smart Contract
📄 **Contract Address:** `0x5A3CaDB539b96cC461a5219C956F8106165BC4ae`
🔍 [View on BscScan](https://bscscan.com/address/0x5A3CaDB539b96cC461a5219C956F8106165BC4ae)

## Features
- ✅ Mint NFTs using BNB (0.001 BNB per mint)
- ✅ 70% automatically donated to LHOPE Fund wallet
- ✅ Permanent IPFS + Filecoin storage via Lighthouse
- ✅ Memorial tributes with images and messages
